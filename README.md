# Python-CalenderTable
Pythonでカレンダーテーブルを作成するテンプレートです。

# Usage

```python
df_calendar = create_calendar_table_spark("2024-01-01", "2024-12-31", first_fiscal_year=1940, start_fiscal_month=6)
display(df_calendar)
```
